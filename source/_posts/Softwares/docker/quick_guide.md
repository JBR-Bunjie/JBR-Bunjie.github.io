# DOCKER QUICK GUIDE

## 忘记教程，即刻上手docker

### 主要依赖：

- Python3.10
- flask
- gunicorn

### 项目内容：

flask官方基础教程结果，见**link1**页面内的**Tutorial**部分。其中的主要内容为：

- 了解flask项目的basic layout
- 建立工厂函数
- 建立并操作数据库
- 了解 `flask: blueprint` 功能
- 了解 `flask: template` 功能
- 项目可安装化
- 基础测试
- 部署
